# 🚀Static Website

A modern, visually stunning, and fully responsive website built using **HTML, CSS, and JavaScript**. The project showcases advanced frontend design techniques including Glassmorphism UI, smooth animations, dark/light mode, responsive layouts, interactive components, and a professional user experience.

## ✨ Features

* 🎨 Modern Glassmorphism Design
* 🌙 Dark & Light Mode Toggle
* 📱 Fully Responsive Layout
* ⚡ Smooth Scroll Animations
* 🔥 Interactive Hero Section
* 📊 Animated Statistics Counters
* 🛠️ Services Section
* 💼 Projects Showcase
* 👥 Team Section
* 📰 Blog Section
* 💰 Pricing Plans
* ❓ Interactive FAQ Accordion
* 📬 Contact Form
* ⬆️ Back To Top Button
* 📱 Mobile Navigation Menu

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome Icons
* Google Fonts (Poppins)

1. Download or clone the repository.
2. Open the project folder.
